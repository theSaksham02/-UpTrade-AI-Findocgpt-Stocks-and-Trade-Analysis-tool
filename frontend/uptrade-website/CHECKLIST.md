## ğŸ†˜ Troubleshooting Quick Reference

### Can't install dependencies?
```bash
rm -rf node_modules .next
pnpm install
```

### Port 3000 in use?
```bash
pnpm dev -p 3001
```

### Build errors?
```bash
# Check Node version
node --version  # Should be 18+

# Clear cache
rm -rf .next
pnpm build
```

### TypeScript errors?
These are configured to be ignored in next.config.mjs during build.

### Animations not working?
Ensure components have `"use client"` at the top.
